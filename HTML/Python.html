Python Introduction

Python is a high-level, general-purpose programming language created by Guido van Rossum and first released in 1991. Its design philosophy emphasizes code readability, notably through the use of significant indentation.

History

Python was conceived in the late 1980s and its implementation began in December 1989 by Guido van Rossum at the Centrum Wiskunde & Informatica (CWI) in the Netherlands. Van Rossum aimed to create a successor to the ABC language that would be capable of exception handling and interfacing with the Amoeba operating system.

Python Working

Python, as an interpreted language, functions through a process involving compilation to bytecode and execution by a virtual machine.

Source Code Creation

A programmer writes Python code in a text editor or Integrated Development Environment (IDE), saving it as a .py file.

Compilation to Bytecode:

When a Python script is executed, the Python interpreter first compiles the source code into an intermediate format called bytecode. This bytecode is platform-independent and stored in .pyc files (or kept in memory for simple scripts). During this compilation, basic syntax checks are performed.

Runtime Operations:

As the PVM executes the bytecode, it handles various runtime operations, including memory management, garbage collection, and interaction with the operating system and hardware resources. Dynamic typing, a core feature of Python, is managed at this stage, where the type of a variable is determined at runtime rather than compile time.

Variables

In Python, variables are essentially named storage locations used to hold data values. Unlike some other programming languages, Python does not require an explicit declaration command to create a variable. Instead, a variable is created the moment a value is assigned to it using the assignment operator (=).

Dynamic Typing

Python is dynamically typed, meaning you do not need to specify the data type of a variable when you declare it. The interpreter infers the type based on the assigned value.

Naming Rules

Must start with a letter (A-z) or an underscore (_). Cannot start with a number. Can contain alphanumeric characters and underscores. Are case-sensitive (e.g., age, Age, and AGE are distinct variables). Cannot be a Python keyword (e.g., if, for, while).

Scope:

Variables have a defined scope, which determines where in the program they can be accessed. Common scopes include local (within a function) and global (accessible throughout the program).

Example

    name = "Alice"  # Assigns the string "Alice" to the variable 'name'
    age = 30        # Assigns the integer 30 to the variable 'age'

     counter = 100          # Creates an integer variable
     miles   = 1000.0       # Creates a floating point variable
     name    = "Zara Ali"   # Creates a string variable

     print (counter)
     print (miles)
     print (name)

Python Data Types

Python includes several built-in data types used to categorize and store different kinds of data. These types determine the operations that can be performed on the data.

Numeric Types int: Represents whole numbers (integers), positive, negative, or zero. float:Represents real numbers with a decimal point (floating-point numbers). 

 num1=20;   #int
 num2=1.24   #float

Sequence Types: str: Represents sequences of characters (strings), enclosed in single or double quotes. List: Represents ordered, changeable collections of items, enclosed in square brackets []. Lists can contain items of different data types. tuple: Represents ordered, unchangeable collections of items, enclosed in parentheses (). Tuples can also contain items of different data types. 

    a= "Hello"
    print(a)   #Hello

    myList=['apple','mango','banana']
    print(myList)

    myTuple=('banana','mango','apple')
    print(myTuple)

Boolean:bool: Represents truth values, either True or False. 

    bool("abc")
    bool(123)
    bool(['apple','cherry','banana'])

Mapping Type: dict: Represents unordered collections of key-value pairs, enclosed in curly braces {}. Keys must be unique and immutable. 

thisDict={
    "brand":"Ford",
    "model":"Mudtang",
    "year":1964
}

Dynamic Typing: Python is dynamically typed, meaning that the data type of a variable is determined at runtime, and a variable can change its data type during execution. The type() function can be used to check the data type of a variable. 

x = 10         # x is an int
print(type(x))

x = "hello"    # x is now a str
print(type(x))

Python Functions

Python functions are block of reusable , organized code designed to perform specific task. They enhance code modularity and promote code reuse.

Defining a Function

Functions are defined using the def keyword, followed by the function name, parentheses and a colon. The code within the function is indented. 

    def greet(name):
     """This function greets the person passed as an argument"""
     print("Hello,{name}!")

Execution

Functions are executed by "calling" them, which involves writing the function name followed by parentheses (and arguments if parameters are defined). 


        greet("Alice") # Calls the greet function

Parameters and Arguments

Functions can accept input values through parameters defined in their signature. When the function is called, the actual values passed are called arguments.

   def add(a, b): # a and b are parameters
        return a + b
          result = add(5, 3) # 5 and 3 are arguments

Return Values:

Functions can return a value using the return keyword. If no return statement is present, the function implicitly returns None.

   def multiply(x, y):
     return x * y
       product = multiply(4, 6) # product will be 24

Python Loops

Python offers two primary types of loops for repetitive execution of code blocks;

for loop

Used for iterating over a sequence (such as a list, tuple, string, or range) or other iterable objects. Executes the code block once for each item in the sequence. 

        for item in sequence:
            # code to be executed for each item
        
         fruits = ["apple", "banana", "cherry"]
          for fruit in fruits:
            print(fruit)

while loop

Used for repeatedly executing a block of code as long as a specified condition remains true. The loop continues until the condition evaluates to False. 

        while condition:
            # code to be executed while condition is true

        count = 0
         while count < 5:
            print(count)
            count += 1

Additionally, Python supports:

Nested Loops

Placing one loop inside another loop. This is useful for tasks involving multi-dimensional structures, like iterating through rows and columns of a 2D array or printing patterns.

Python Objects

In Python, an object is a fundamental building block of the language's object-oriented programming paradigm. Essentially, an object is an instance of a class, which serves as a blueprint or template defining the object's structure and behavior.

Key characteristics of Python objects:

Instance of a Class:

Every object is created from a specific class, inheriting its defined attributes (data) and methods (functions). For example, if you have a Car class, my_car = Car("red", "sedan") creates an object my_car that is an instance of the Car class.

Attributes:

These are variables or data associated with an object, representing its state or properties. In the Car example, color and type would be attributes of the my_car object.

Methods:

These are functions defined within a class that operate on the object's attributes or perform actions related to the object. A Car object might have methods like start_engine() or drive().

Identity, Type, and Value:

Every object in Python has a unique identity (its memory address), a specific type (the class it belongs to), and a value (the data it holds).

Everything is an Object:

In Python, almost everything you interact with, including numbers, strings, lists, dictionaries, and even functions, is an object of a particular built-in class.

Creating Objects:

Objects are created by "instantiating" a class, typically by calling the class name as if it were a function. The __init__ method, also known as the constructor, is a special method within a class that is automatically called when a new object is created. It's used to initialize the object's attributes.

class Dog:
    def __init__(self, name, breed):
        self.name = name
        self.breed = breed

    def bark(self):
        print(f"{self.name} barks!")

# Creating an object (instance) of the Dog class
my_dog = Dog("Buddy", "Golden Retriever")

# Accessing attributes
print(my_dog.name)
print(my_dog.breed)

# Calling a method
my_dog.bark()

Python Classes

In Python, a class serves as a blueprint or a template for creating objects. It encapsulates data (attributes) and functions (methods) that operate on that data into a single, cohesive unit. This approach is fundamental to Object-Oriented Programming (OOP) in Python, promoting code organization, reusability, and modularity.

Class Definition

Classes are defined using the class keyword, followed by the class name (conventionally capitalized using PascalCase).

        class MyClass:
         # Class body
           pass # Placeholder if the class body is empty

Objects (Instances)

An object is a concrete instance of a class. You create objects by "calling" the class name, similar to a function.

      my_object = MyClass()

Attributes

Attributes are variables associated with a class or its instances. Instance Attributes: Unique to each object, defined within the __init__ method using self.attribute_name. Class Attributes: Shared by all instances of the class, defined directly within the class body.

Methods

Methods are functions defined within a class that operate on the object's data. They always take self as their first parameter, which refers to the instance of the class.

     class Dog:
        def __init__(self, name, age):
            self.name = name
            self.age = age

        def bark(self):
            return f"{self.name} says woof!"

Example

 class Car:
    # Class attribute
    wheels = 4

    def __init__(self, make, model, year):
        # Instance attributes
        self.make = make
        self.model = model
        self.year = year

    def display_info(self):
        return f"This is a {self.year} {self.make} {self.model} with {self.wheels} wheels."

        # Create objects (instances)
        car1 = Car("Toyota", "Camry", 2023)
        car2 = Car("Honda", "Civic", 2022)

        # Access attributes and call methods
        print(car1.display_info())
        print(car2.make)
        print(Car.wheels)

Python JSON

Python provides a built-in module called json for working with JSON (JavaScript Object Notation) data. JSON is a lightweight data-interchange format that is widely used for storing and exchanging data, particularly in web applications and APIs.

Serialization

json.dumps(): Encodes a Python object into a JSON formatted string. json.dump(): Encodes a Python object and writes it to a file-like object.

     import json

    data = {
        "name": "Alice",
        "age": 30,
        "isStudent": False,
        "courses": ["Math", "Science"]
    }

    # Serialize to a JSON string
    json_string = json.dumps(data, indent=4) # indent for pretty-printing
    print(json_string)

    # Serialize to a JSON file
    with open("data.json", "w") as f:
        json.dump(data, f, indent=4)

Deserialization

json.loads(): Decodes a JSON formatted string into a Python object. json.load(): Reads a JSON formatted file and decodes it into a Python object.

        import json

    json_string = '{"name": "Bob", "age": 25, "isStudent": true}'

    # Deserialize from a JSON string
    python_object = json.loads(json_string)
    print(python_object)
    print(type(python_object))

    # Deserialize from a JSON file
    with open("data.json", "r") as f:
        loaded_data = json.load(f)
    print(loaded_data)

Python Math

Python offers several ways to perform mathematical operations, ranging from basic arithmetic to complex scientific calculations.

Built-in Math Functions

Python includes several built-in functions for common mathematical tasks min() and max(): Return the lowest and highest values in an iterable, respectively. abs(): Returns the absolute (positive) value of a number. pow(x, y): Returns x raised to the power of y (x^y).

2. The math Module

For more advanced mathematical functions and constants, Python provides the built-in math module. This module must be imported to be used:

Once imported, you can access its functions and constants

math.pi: Represents the mathematical constant pi (approximately 3.14159). math.e: Represents Euler's number (approximately 2.71828).

Functions

math.sqrt(x): Returns the square root of x. math.ceil(x): Rounds x upwards to the nearest integer. math.floor(x): Rounds x downwards to the nearest integer. math.sin(x), math.cos(x), math.tan(x): Trigonometric functions (expecting input in radians). math.log(x, base): Returns the logarithm of x with the specified base. math.exp(x): Returns e raised to the power of x. And many more for hyperbolic functions, factorials, GCD, etc.

Example

# Absolute value
print(abs(-7.25)) # Output: 7.25

# Minimum value
print(min(5, 10, 2)) # Output: 2

# Maximum value
print(max(5, 10, 2)) # Output: 10

# Rounding
print(round(3.14159, 2)) # Output: 3.14

import math

# Square root
print(math.sqrt(64)) # Output: 8.0

# Ceiling (rounds up to the nearest integer)
print(math.ceil(3.14)) # Output: 4

# Floor (rounds down to the nearest integer)
print(math.floor(3.99)) # Output: 3

# Pi constant
print(math.pi) # Output: 3.141592653589793

# Sine of an angle (in radians)
print(math.sin(math.radians(90))) # Output: 1.0 (approx)

# Logarithm (base e)
print(math.log(10)) # Output: 2.302585092994046

# Power function (x to the power of y)
print(math.pow(2, 4)) # Output: 16.0
